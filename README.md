---
- Clone this respository
> `git clone https://github.com/devsdenepal/andro-predom`
- Change the current directory to andro-predom
> `cd andro-predom`
- CHMOD
> `chmod +x  *`

> `sudo ./install.sh`

> `./andro-predom`
